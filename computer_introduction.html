Basic Computer Introduction

What is a Computer?

A computer is an electronic device that processes input and produces output. It is used to type documents, browse the internet, send emails, play music, and much more. A screen reader user can control a computer completely with the keyboard.

Versions of Computers

Computers have evolved over time. Here are the major types:

  • Mainframe Computers: Large and powerful, used in big organizations.
  • Desktop Computers: Commonly used at home and offices.
  • Laptops: Portable computers for mobility.
  • Tablets and Smartphones: Smaller touch-screen based computers.

Extra Information

Modern computers work using a combination of hardware (the physical parts) and software (the programs). People with visual impairments can use screen readers like NVDA or JAWS to interact with the computer efficiently.
