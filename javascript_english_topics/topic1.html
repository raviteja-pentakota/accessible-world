Coding with Confidence: Your First JavaScript code with Screen Readers [English]

Welcome to the first episode of our JavaScript tutorial series designed to be accessible with screen readers!

In this lesson, we'll introduce you to the basics of JavaScript and how it can be used to modify web page content dynamically. By working with headings and paragraphs, you'll see how JavaScript can bring interactivity and flexibility to your HTML pages.

By the end of this video, you'll be able to:

  • Understand the role of JavaScript in web development.
  • Modify the text content of headings and paragraphs using JavaScript.
  • Use basic JavaScript syntax to select HTML elements and change their properties.
  • Enhance accessibility by ensuring dynamic content changes are compatible with screen readers.

This tutorial is perfect for beginners who want to start learning JavaScript while keeping accessibility in mind. No prior programming experience is needed!

For your reference, a "View Code" button will be provided below the video, allowing you to view the practice code used in the lesson.

Let's dive into JavaScript and start making your web pages interactive!

Topic 1 example code:

All the below content is the JavaScript code for the first topic:


<!DOCTYPE html>
<html>
<head>
<title>First JS Activity</title>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
</head>
<body>
<h1 id="h1"></h1>
<p id="p1"></p>
<script>
document.getElementById("h1").innerHTML = "Welcome!";
document.getElementById("p1").innerHTML = "Please subscribe Accessible World Tutorials <br> Youtube channel...";
</script>
</body>
</html>
